# Springboot-CRUD-App
-----------------------

## About
-----------
A Simple SpringBoot CRUD App. 
There are five endpoints; 
	**GET/api/employees**
	**POST/api/employees**
	**GET/api/employees/{id}**
	**PUT/api/employees{id}**
	**DELETE/api/employees{id}**


## Technology
	1. Java
	2. Springboot
	3. mySQL
