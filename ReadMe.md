# Springboot-CRUD-App
-----------------------

## About
-----------
A Simple SpringBoot CRUD App for an Employee Management system. 
There are five endpoints; \
	1. GET/api/employees \
	2. POST/api/employees \
	3. GET/api/employees/{id} \
	4. PUT/api/employees{id} \
	4. DELETE/api/employees{id}


## Technology
-------------
	1. Java
	2. Springboot
	3. mySQL
